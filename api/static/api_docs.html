AutoMATA 训练任务管理系统 API 文档

基础信息

  • 基础URL: http://localhost:8000/api
  • 文档: Swagger UI | ReDoc
  • 健康检查: 根路径

用户管理 API

POST /api/users/
创建新用户
请求示例:
curl -X POST http://localhost:8000/api/users/ \
  -H "Content-Type: application/json" \
  -d '{"username":"testuser","email":"test@example.com","password":"password123"}'
响应示例:
{"id": 1, "username": "testuser", "email": "test@example.com", "is_active": true, "created_at": "2026-02-05T19:35:24"}
GET /api/users/
获取所有用户列表
curl http://localhost:8000/api/users/
GET /api/users/{user_id}
根据ID获取特定用户
PUT /api/users/{user_id}
更新用户信息
DELETE /api/users/{user_id}
删除用户

训练任务管理 API

POST /api/training/tasks
创建新的训练任务
请求示例:
curl -X POST http://localhost:8000/api/training/tasks \
  -H "Content-Type: application/json" \
  -d '{"task_name":"测试任务","model_type":"supervised","language":"python","parameters":"{\"learning_rate\": 0.01}","dataset_path":"/data/dataset.csv","created_by":1}'
GET /api/training/tasks
获取训练任务列表
curl http://localhost:8000/api/training/tasks
GET /api/training/tasks/{task_id}
根据ID获取特定训练任务
PUT /api/training/tasks/{task_id}
更新训练任务
DELETE /api/training/tasks/{task_id}
删除训练任务
GET /api/training/tasks/{task_id}/logs
获取特定任务的训练日志
POST /api/training/logs
添加训练日志

状态说明

  • 任务状态: pending(待处理), running(运行中), completed(已完成), failed(失败)
  • 模型类型: supervised(监督学习), unsupervised(无监督学习)
  • 语言: python, r
